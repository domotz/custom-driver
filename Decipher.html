Namespace: Decipher

Decipher

Object for encrypting data. Returned by D.crypto.createDecipher. Decipher objects are streams that are both readable and writable. The written enciphered data is used to produce the plain-text data on the the readable side. The legacy update and final methods are also supported.

Methods

(static) final(outputEcodingopt) → {string|Buffer}

Returns any remaining plaintext which is deciphered. The Decipher object can not be used after final() method has been called. If no encoding is provided, then a buffer is returned.
Parameters:
Name Type Attributes Description
outputEcoding binary | base64 | hex <optional>
 The outputEncoding specifies the output format of the enciphered data, and can be 'binary', 'base64' or 'hex'. If no encoding is provided, then a buffer is returned.
Returns:
- Returns any remaining deciphered data
Type
string | Buffer
Example
D.crypto.createDecipher('sha1', 'key', 'iv').final('binary')

(static) setAutoPadding(autoPaddingopt)

You can disable auto padding if the data has been encrypted without standard block padding to prevent decipher.final from checking and removing it. Can only work if the input data's length is a multiple of the ciphers block size. You must call this before streaming data to decipher.update.
Parameters:
Name Type Attributes Description
autoPadding bool <optional>
 The auto padding boolean flag.
Example
D.crypto.createDecipher('sha1', 'key', 'iv').setAutoPadding(false)

(static) update(data, inputEncodingopt, outputEcodingopt) → {string|Buffer}

Updates the decipher with data, which is encoded in 'binary', 'base64' or 'hex'. If no encoding is provided, then a buffer is expected.
Parameters:
Name Type Attributes Description
data string | Buffer The data object. If data is a Buffer then inputEncoding is ignored.
inputEncoding binary | ascii | utf8 <optional>
 The input encoding. If data is a Buffer then inputEncoding is ignored.
outputEcoding binary | base64 | hex <optional>
 The output encoding specifies the output format of the enciphered data, and can be 'binary', 'base64' or 'hex'. If no encoding is provided, then a buffer is returned.
Returns:
- Returns the deciphered data
Type
string | Buffer
Example
D.crypto.createDecipher('sha1', 'key', 'iv').update('data', 'binary', 'binary')
