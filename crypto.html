Namespace: crypto

crypto

The Custom Driver Crypto object. Exposes Crypto related libraries. Contains utility functions that offer ways of encapsulating secure credentials to be used as part of a secure HTTPS net or http connection. It also offers a set of wrappers for OpenSSL's hash, hmac, cipher, decipher methods.
