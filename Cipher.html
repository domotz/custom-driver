Namespace: Cipher

Cipher

Object for encrypting data. Returned by D.crypto.createCipher. Cipher objects are streams that are both readable and writable. The written plain text data is used to produce the encrypted data on the readable side. The legacy update and final methods are also supported.

Methods

(static) final(outputEcodingopt) → {string|Buffer}

Returns any remaining enciphered contents, with outputEncoding being one of: 'binary', 'base64' or 'hex'. If no encoding is provided, then a buffer is returned. The Cipher object cannot be used after final() method has been called.
Parameters:
Name Type Attributes Description
outputEcoding binary | base64 | hex <optional>
 The outputEncoding specifies the output format of the enciphered data, and can be 'binary', 'base64' or 'hex'. If no encoding is provided, then a buffer is returned.
Returns:
- Any remaining enciphered contents
Type
string | Buffer
Example
D.crypto.createCipher('sha1', 'key', 'iv').final('binary')

(static) setAutoPadding(autoPaddingopt)

You can disable automatic padding of the input data to block size. If autoPadding is false, the length of the entire input data must be a multiple of the cipher's block size or final will fail. Useful for non-standard padding, e.g. using 0x0 instead of PKCS padding. You must call this before cipher.final.
Parameters:
Name Type Attributes Description
autoPadding bool <optional>
 The auto padding boolean flag.
Example
D.crypto.createCipher('sha1', 'key', 'initialization vector').setAutoPadding(false)

(static) update(data, inputEncodingopt, outputEcodingopt) → {string|Buffer}

Updates the cipher with data, the encoding of which is given in inputEcoding and can be 'utf8', 'ascii' or 'binary'. If data is a Buffer then inputEncoding is ignored.
Parameters:
Name Type Attributes Description
data string | Buffer The data object. If data is a Buffer then inputEncoding is ignored.
inputEncoding binary | ascii | utf8 <optional>
 The input encoding. If data is a Buffer then inputEncoding is ignored.
outputEcoding binary | base64 | hex <optional>
 The output encoding specifies the output format of the enciphered data, and can be 'binary', 'base64' or 'hex'. If no encoding is provided, then a buffer is returned.
Returns:
- Returns the enciphered contents, and can be called many times with new data as it is streamed.
Type
string | Buffer
Example
D.crypto.createCipher('sha1', 'key', 'iv').update('data', 'binary', 'binary')
